# Flare Predictor — Система раннего предупреждения для факельной установки НПЗ

Курсовая работа: 3D-модель факельной установки (Blender/Unity) с системой мониторинга и прогнозирования аварийных ситуаций на основе логистической регрессии.

## Структура проекта

```
flare-predictor/
├── data/           — исходные данные (CSV)
├── stats/          — статистический анализ (Python-скрипты, графики)
├── blender/        — 3D-модель факельной установки (.blend)
├── unity/          — Unity-проект (сцена, скрипты, анимации)
│   └── assets/
│       ├── scripts/   — C#-скрипты
│       ├── models/    — импортированные модели
│       └── textures/  — текстуры и материалы
├── report/         — итоговый отчёт
│   └── images/     — скриншоты и рендеры
└── README.md
```

## Ключевые параметры

| Параметр  | Норма |
|-----------|-------|
| P_flare   | 0.004–0.016 МПа |
| Q_flare   | ≥1.5 м³/ч |
| P_purge   | 0.3–0.6 МПа |
| Q_purge   | 20–50 м³/ч |
| T_flame   | 800–1200 °C |

## Аварийные ситуации

- **Отрыв пламени**: P_purge < 0.28 МПа И Q_purge < 22 м³/ч
- **Хлопок**: P_flare > 0.022 МПа с последующим падением

## Статистическая модель (логистическая регрессия)

P(отрыв) = 1 / (1 + exp(-(b0 + b1·P_purge + b2·Q_purge + b3·P_flare)))
